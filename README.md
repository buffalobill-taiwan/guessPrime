# 猜質數 (guessPrime)

這是一個簡單的 HTML 遊戲，玩家需要判斷畫面上顯示的數字是質數還是非質數（半質數）。

## 遊戲玩法

1. 畫面會顯示一個數字及其位元數 (Bits)。
2. 玩家需要點擊「是質數」或「不是質數」按鈕進行猜測。
3. **正確**：位元數會增加 1 (最高 64 bits)，表示難度提升。
4. **錯誤**：位數會減少 1 (最低 8 bits)，表示難度降低。
5. 如果數字是**半質數** (p × q)，遊戲會顯示其因數分解結果。

## 技術細節

- **開發技術**: HTML5, JavaScript (BigInt), CSS (UIKit CSS framework), Audio API (合成音訊回饋)。
- **演算法**: 使用 **Miller-Rabin 素性測試** 演算法來判斷數字是否為質數。
- **數字生成**: 
  - 50% 機率生成質數。
  - 50% 機率生成半質數 (兩個較小質數的乘積)，且會顯示其因數分解結果（由小至大）。
- **特色**: 
  - 支援高達 64 位元的超大數字計算。
  - 具備動態視覺回饋與音訊輔助功能。

## 如何執行

只需使用任何現代瀏覽器直接打開 `index.html` 即可開始遊戲，無需任何建置步驟。

## 專案結構

- `index.html`: 遊戲的主介面。
- `js/game.js`: 遊戲核心邏輯與控制流程。
- `js/math-utils.js`: 數學工具函式庫（包含素性測試與數字生成）。
- `css/`: 包含樣式表與 UIKit CSS 框架。
